# Student Daily Task Manager

A web-based task management application designed to help students organize, track, and manage their daily tasks effectively.

## Features

* Add daily tasks
* Mark tasks as completed
* Delete tasks
* Easy task management
* Clean and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose

This project was developed to help students stay organized, improve productivity, and manage their daily activities efficiently.
